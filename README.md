# MindAlertsTask

Security alerts dashboard built with React, TypeScript, Vite, React Router, React Query and MUI.

- GitHub: https://github.com/yoad100/MindAlertsTask
- Live demo (Vercel): https://mind-alerts-task.vercel.app/

## Run locally

```bash
npm install
npm run dev
```

## Tests

```bash
npm run test
```
